## Overview

Fin-agents is a sophisticated multi-model agent system designed for autonomous financial advising and data analysis. Built primarily with Django, FastAPI, TypeScript, and Google Agent Development Kit (ADK), this project enables seamless backend integrations and advanced financial insights.

## Features

- Multi-model agents for autonomous financial advising and data analysis.
- Backend integrations via MCP connecting reinforcement learning and trading algorithm models.
- Retrieval-Augmented Generation (RAG) system to enhance financial insights using curated trading and investment literature.
- Real-time stock market visualization and sentiment analytics dashboards.

## Technologies Used

- Python (Django, FastAPI)
- TypeScript
- Google Agent Development Kit (Google ADK)
- Reinforcement Learning Models
- MCP Integration
- Dashboard Visualization tools

## Repository Structure

- Core agent system and financial data analysis modules.
- Integration components for connecting machine learning and trading models.
- API endpoints and services built with FastAPI.
- Frontend components using TypeScript.
